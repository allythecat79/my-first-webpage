Yarn

First you need yarn. A lot of people start with a washcloth as their first project. For a washcloth you need an inexpensive cotton yarn. As your projects get more varied, you'll want different types of yarn.

Here are a few yarns to start with.

  1. Wool yarn
    • Fine wool
    • Medium wool
    • Long wool
    • Double wool
  2. Cashmere yarn
  3. Alpaca yarn
  4. Multi fiber yarn

Stitches

There are so many types of stitches that make so many beautiful patterns.

Here are a few stitches you might use as a beginner.

  1. Knit stitch
  2. Purl stich
  3. Stockinette stitch
  4. Seed stitch
  5. Garter stitch
